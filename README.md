# 📱 Mobile Sales Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze mobile sales performance. It provides valuable insights into sales trends, customer behavior, and business performance using interactive visualizations.

---

## 🎯 Objectives

- Analyze total sales performance
- Track monthly sales trends
- Compare mobile brands and models
- Understand customer ratings
- Analyze payment methods
- Identify top-performing cities

---

## 🛠️ Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Key KPIs

- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales

---

## 📈 Dashboard Features

- KPI Cards
- Sales by City
- Sales by Mobile Brand
- Monthly Sales Trend
- Customer Ratings
- Payment Method Analysis
- Interactive Filters
